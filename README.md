# GifsApp

A learning project focused on building unit tests with Vitest.

## Overview

This repository is a small React + TypeScript app for searching GIFs, designed to help developers learn and practice unit testing.

- Uses Vitest for unit testing.
- Implements React components with TypeScript.
- Includes API integration with Giphy.

## Purpose

The goal is to practice writing robust, maintainable unit tests in a frontend project while keeping the app simple and easy to explore.

## Getting Started

```bash
pnpm install
pnpm test
```

## Project Structure

- `src/`: application source code.
- `src/gifs/`: GIF-specific logic, API calls, actions, and components.
- `src/shared/`: shared UI components.

## Notes

This project is intended for testing practice rather than production deployment, so the focus is on testability and learning.
